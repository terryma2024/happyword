# V0.4.6 — 更多主题区域 (Implementation Plan)

> Status: in progress  
> Branch: `feat/v0.4-deep-learning-and-spelling`  
> Author: assistant (autonomous)

## Goal

Give "今日冒险" room to keep feeling fresh by adding two new themed
regions backed by two new word categories. The kid sees a longer
region picker (3 → 5) and the dictionary nearly doubles in coverage
(30 → 50 words).

## Out of scope

- New boss SVGs / codex entries: V0.3.8 already shipped 7 bosses;
  V0.4.6 reuses them across regions rather than expanding the
  codex. Adding new boss art is content backlog (V0.5+).
- Per-region question pacing (HP curves, monster count): all new
  regions reuse the standard 5-slot
  `Normal / Spelling / Review / Elite / Boss` plan.
- Localised UI for category labels in non-Simplified-Chinese
  locales: the project is zh-CN only today.

## Decisions

1. **New categories.** `animal` (10 land animals) and `ocean` (10
   sea creatures). Both stay in difficulty=1 with the existing
   word-data shape; no new fields.
2. **New regions.**
   - `animal-safari` (动物乐园) — themeWordCategories=['animal'],
     `bossCandidates=[6, 9]` (Phoenix + Unicorn, both fairy-tale
     creatures) — Phoenix is shared with fruit-forest and Unicorn
     stays exclusively here after the migration in (3).
   - `ocean-realm` (深海王国) — themeWordCategories=['ocean'],
     `bossCandidates=[10]` (Kraken). This fulfils V0.3.8's
     standing promise: "Kraken will migrate to a future ocean
     region in v0.4". Keeps the ocean roster lean for now; future
     V0.4.x can add a 12th boss.
3. **Boss migration.** `home-cottage` drops Kraken from
   `bossCandidates=[9,10]` to `[9]` (Unicorn only). The "single
   boss" rotation pool collapses to one candidate, which is fine —
   the rotation is by `hash(regionId+dayKey) % candidates.length`
   and modulo by 1 always returns `[0]`. The cottage has fewer
   visual variations as a result, but this is the cleanest path
   to giving Kraken a thematically appropriate home.
4. **Boss sharing.** V0.3.8's `bossCandidatesUnionCoversAllSeven`
   uniqueness invariant relaxes in V0.4.6. We keep the union
   coverage requirement (every boss is reachable from some
   region), but multiple regions can list the same boss in their
   rotation pool. Update the test to assert "every catalog index
   4..10 appears in at least one region's `bossCandidates`".
5. **Region picker UI.** With 5 regions the inline `Row` overflows
   typical phone widths (5 × ~80 dp + 4 × 10 dp ≈ 440 dp). Wrap
   the picker in a horizontal `Scroll` so longer rosters never
   clip; the `RegionChip_*` ids stay tappable.
6. **Category labels.** Add `animal → 动物`, `ocean → 海洋` to
   `LearningReportBuilder.describeCategory`. Leaves the report
   page automatically picking up the new categories.
