# V0.6.5 Three-layer Pack Model Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Promote each "battle scene" to a first-class `Pack` sourced from three layers (built-in / global / family); let users pick up to 5 active packs with optional pin and 3-perfect-run auto-rotation; retire the legacy single-`WordRepository` pipeline that lets server test words leak into Today mode.

**Architecture:** Mirror the existing family-pack server stack (`FamilyPackDefinition / FamilyPackDraft / FamilyWordPack / FamilyPackPointer` -> `parent_family_pack` + `child_family_pack` routers) with a parallel global-pack stack and a public, anonymous `latest.json` endpoint. On the client, introduce a `Pack` model and `PackLibrary` that unions the three layers, a `PackSelectionService` that owns active-set + pin + perfect-score state in a dedicated preferences slot, and rewire `BattlePage / TodayAdventureBuilder / HomePage / ConfigPage` to consume packs directly.

**Tech Stack:** FastAPI, Beanie/MongoDB, Pydantic v2, pytest + httpx ASGI tests, `uv`. ArkTS (HarmonyOS NEXT), Hvigor, ohosTest, ohpm.

## Scope Boundary

This plan implements V0.6.5 client minor release in full. It does **not**:

- Migrate v0.6.x clients off the legacy `/api/v1/packs/latest.json` endpoint (kept untouched for backwards compat).
- Add an admin web UI for cross-category global packs (CRUD via JSON only in this release).
- Cloud-sync the local active-set / pin / perfect-score state.
- Replace the `LearningRecorder` or any audio / TTS / coin / wishlist subsystems.

The companion design doc is [docs/superpowers/specs/2026-05-09-v0.6.5-three-layer-pack-model-design.md](docs/superpowers/specs/2026-05-09-v0.6.5-three-layer-pack-model-design.md). Read it before starting Task 1.

## Task 2: family_pack_service — accept custom pack_id + scene + global publish guard

**Files:**
- Modify: `server/app/services/family_pack_service.py`
- Create: `server/tests/test_family_pack_service_v065.py`

This task extends the existing service in five small ways. None of them break v0.6.4 callers:

1. `create_definition` accepts optional `pack_id: str | None = None` (default keeps current `_gen_pack_id()` behaviour) and optional `scene: dict[str, Any] | None = None` (defaults to `{}`).
2. `patch_definition` accepts optional `scene: dict[str, Any] | None = None`.
3. `MergedSlice` carries `description: str | None` and `scene: dict[str, Any]`. `collect_merged` populates them from the live definition.
4. `publish` drops draft words whose `category == "test"` **only when** `family_id == GLOBAL_PACK_FAMILY_ID`. Family packs (real `family_id`) keep the legacy permissive behaviour.
5. `_build_entry` accepts a new `source == "global"` *fallback* path: when called for a definition whose `family_id == GLOBAL_PACK_FAMILY_ID`, the strict `custom_prefix` check on `word_id` is bypassed (admins are not required to prefix global pack words with `fam-__global_-`). The other source modes are unchanged.

> Note: the existing service's `CustomIdContract(family_id=...).prefix` for the global sentinel evaluates to `fam-__global_-`, which is awkward for admin authoring of global packs. Item (5) above fixes this by short-circuiting the prefix enforcement when the underlying family is the global sentinel. The `_build_entry` signature stays compatible — only the validation behaviour changes.

```python
# server/app/services/global_pack_service.py
"""V0.6.5 — Thin wrapper around family_pack_service for global packs.

Hides the `family_id=GLOBAL_PACK_FAMILY_ID` sentinel from admin-side
callers and forces a `gpk-` prefix for newly minted pack ids so the
source layer is recognizable. All persistence lives in the underlying
family-pack stack; this module is purely sugar.
"""

from __future__ import annotations

import secrets
from typing import Any

from app.models.family_pack_definition import FamilyPackDefinition
from app.models.family_pack_draft import FamilyPackDraft
from app.models.family_pack_pointer import FamilyPackPointer
from app.models.family_word_pack import FamilyWordPack
from app.services import family_pack_service as fps

GLOBAL_PACK_FAMILY_ID = fps.GLOBAL_PACK_FAMILY_ID
GlobalPackError = fps.FamilyPackError
PackNotFound = fps.PackNotFound
EmptyPack = fps.EmptyPack
WordLimitExceeded = fps.WordLimitExceeded
NoPreviousVersion = fps.NoPreviousVersion
InvalidPayload = fps.InvalidPayload
MergedSlice = fps.MergedSlice


def _gen_pack_id() -> str:
    return f"gpk-{secrets.token_hex(4)}"


async def create_definition(
    *,
    name: str,
    admin_id: str,
    description: str | None = None,
    scene: dict[str, Any] | None = None,
    pack_id: str | None = None,
) -> FamilyPackDefinition:
    return await fps.create_definition(
        family_id=GLOBAL_PACK_FAMILY_ID,
        parent_user_id=admin_id,
        name=name,
        description=description,
        scene=scene or {},
        pack_id=pack_id or _gen_pack_id(),
    )


async def get_definition(*, pack_id: str) -> FamilyPackDefinition:
    return await fps.get_definition_for_family(family_id=GLOBAL_PACK_FAMILY_ID, pack_id=pack_id)


async def list_definitions(*, include_archived: bool = False) -> list[FamilyPackDefinition]:
    return await fps.list_definitions(family_id=GLOBAL_PACK_FAMILY_ID, include_archived=include_archived)


async def patch_definition(
    *,
    pack_id: str,
    admin_id: str,
    name: str | None = None,
    description: str | None = None,
    scene: dict[str, Any] | None = None,
) -> FamilyPackDefinition:
    # Note: family_pack_service.patch_definition does not currently accept a
    # parent_user_id parameter (audit trail is recorded elsewhere). If that
    # changes, thread admin_id through here. For now admin_id is unused but
    # kept in the wrapper signature for symmetry with create_definition.
    _ = admin_id
    return await fps.patch_definition(
        family_id=GLOBAL_PACK_FAMILY_ID,
        pack_id=pack_id,
        name=name,
        description=description,
        scene=scene,
    )


async def archive(*, pack_id: str) -> FamilyPackDefinition:
    return await fps.archive(family_id=GLOBAL_PACK_FAMILY_ID, pack_id=pack_id)


async def get_or_create_draft(*, pack_id: str, admin_id: str) -> FamilyPackDraft:
    definition = await get_definition(pack_id=pack_id)
    return await fps.get_or_create_draft(definition=definition, parent_user_id=admin_id)


async def upsert_draft_word(
    *, pack_id: str, admin_id: str, entry: dict[str, Any]
) -> FamilyPackDraft:
    """Wire `entry` (the admin/migration-friendly camelCase shape) onto
    family_pack_service's payload contract (source='global' + snake_case)."""
    definition = await get_definition(pack_id=pack_id)
    word_id = entry["id"]
    payload: dict[str, Any] = {
        "source": entry.get("source", "global"),
        "word": entry.get("word"),
        "meaning_zh": entry.get("meaningZh") or entry.get("meaning_zh"),
        "category": entry.get("category"),
        "difficulty": entry.get("difficulty"),
    }
    for src_camel, src_snake in (
        ("distractors", "distractors"),
        ("exampleEn", "example_en"),
        ("exampleZh", "example_zh"),
        ("illustrationUrl", "illustration_url"),
        ("audioUrl", "audio_url"),
    ):
        v = entry.get(src_camel) or entry.get(src_snake)
        if v is not None:
            payload[src_snake] = v
    return await fps.upsert_draft_word(
        definition=definition, word_id=word_id, payload=payload, parent_user_id=admin_id,
    )


async def remove_draft_word(
    *, pack_id: str, admin_id: str, word_id: str
) -> FamilyPackDraft:
    definition = await get_definition(pack_id=pack_id)
    return await fps.remove_draft_word(
        definition=definition, word_id=word_id, parent_user_id=admin_id,
    )


async def publish(
    *, pack_id: str, admin_id: str, notes: str | None = None
) -> FamilyWordPack:
    definition = await get_definition(pack_id=pack_id)
    return await fps.publish(definition=definition, parent_user_id=admin_id, notes=notes)


async def rollback(*, pack_id: str) -> FamilyPackPointer:
    definition = await get_definition(pack_id=pack_id)
    return await fps.rollback(definition=definition)


async def list_versions(*, pack_id: str) -> list[FamilyWordPack]:
    definition = await get_definition(pack_id=pack_id)
    return await fps.list_versions(definition=definition)


async def collect_merged() -> tuple[list[fps.MergedSlice], str]:
    return await fps.collect_merged(family_id=GLOBAL_PACK_FAMILY_ID)
```

```python
def _serialize_definition(d: FamilyPackDefinition) -> GlobalPackDefinitionOut:
    # Map FamilyPackDefinition.created_by_parent_id -> wire field created_by_admin_id
    # for global packs (see spec §5.3 — same field, role-named for the admin context).
    return GlobalPackDefinitionOut(
        pack_id=d.pack_id,
        name=d.name,
        description=d.description,
        scene=d.scene,

    return HTTPException(status_code=http_status, detail={"error": {"code": code, "message": message}})
```

```python
# server/app/routers/public_global_pack.py
"""V0.6.5 — public anonymous endpoint for global packs.

Mirrors `/api/v1/child/family-packs/latest.json` but with NO Bearer auth —
global content is public. Vercel deployment-protection still gates preview
deploys; production stays open. Returns 204 when there are no published
global packs (cold prod state pre-migration).
"""

from __future__ import annotations

from datetime import UTC, datetime

from fastapi import APIRouter, Header, Response, status

from app.schemas.global_pack import GlobalPackEntryInMerged, GlobalPacksLatestOut
from app.services import global_pack_service as svc

router = APIRouter(prefix="/api/v1/public", tags=["public-global-pack"])


def _max_schema_version(slices: list[svc.MergedSlice]) -> int:
    # FAMILY_PACK_SCHEMA_VERSION is reused — global and family packs share the wire schema.
    from app.services.family_pack_service import FAMILY_PACK_SCHEMA_VERSION
    return max((s.schema_version for s in slices), default=FAMILY_PACK_SCHEMA_VERSION)


@router.get("/global-packs/latest.json", response_model=None)
async def get_global_packs_latest(
    response: Response,
    if_none_match: str | None = Header(default=None),
) -> GlobalPacksLatestOut | Response:
    slices, etag = await svc.collect_merged()
    if not slices:
        return Response(status_code=status.HTTP_204_NO_CONTENT)
    if if_none_match is not None and if_none_match.strip() == etag:
        return Response(status_code=status.HTTP_304_NOT_MODIFIED, headers={"ETag": etag})
    response.headers["ETag"] = etag
    response.headers["Cache-Control"] = "public, max-age=60"
    return GlobalPacksLatestOut(
        schema_version=_max_schema_version(slices),
        merged_at=datetime.now(tz=UTC),
        packs=[
            GlobalPackEntryInMerged(
                pack_id=s.pack_id,
                name=s.name,
                description=s.description,
                scene=s.scene,
                version=s.version,
                schema_version=s.schema_version,
                published_at=s.published_at,
                words=s.words,
            )
            for s in slices
        ],
    )


@router.head("/global-packs/latest.json", response_model=None)
async def head_global_packs_latest() -> Response:
    _, etag = await svc.collect_merged()
    return Response(status_code=200, headers={"ETag": etag})
```

```typescript
// entry/src/main/ets/services/BuiltinPackLoader.ets
import { common } from '@kit.AbilityKit';
import { util } from '@kit.ArkTS';
import { BusinessError } from '@ohos.base';

import { Pack, SceneMetadata } from '../models/Pack';
import { WordEntry, ExampleSentence } from '../models/WordEntry';
import { MonsterPlan, MonsterSlot } from '../models/AdventureRegion';
import { MonsterKind, monsterIndexForKind } from '../data/MonsterCatalog';

const BUILTIN_PACK_FILES: string[] = [
  'data/builtin/fruit-forest.json',
  'data/builtin/school-castle.json',
  'data/builtin/home-cottage.json',
  'data/builtin/animal-safari.json',
  'data/builtin/ocean-realm.json',
];

interface RawSlot {
  kind?: string;
  catalogIndex?: number;
}

interface RawScene {
  bgPrimary?: string;
  bgAccent?: string;
  bossName?: string;
  bossCandidates?: number[];
  monsterPlan?: RawSlot[];
  storyZh?: string;
}

interface RawWord {
  id?: string;
  word?: string;
  meaningZh?: string;
  category?: string;
  difficulty?: number;
  distractors?: string[];
  example?: { en?: string; zh?: string };
  illustrationUrl?: string;
  audioUrl?: string;
}

interface RawBuiltinPack {
  pack_id?: string;
  name?: string;
  labelZh?: string;
  scene?: RawScene;
  words?: RawWord[];
}

export class BuiltinPackLoader {
  /** Pure parser used by tests; no IO. */
  static parsePackJson(jsonText: string): Pack | undefined {
    let raw: RawBuiltinPack;
    try {
      raw = JSON.parse(jsonText) as RawBuiltinPack;
    } catch (_err) {
      return undefined;
    }
    if (raw === null || raw === undefined || typeof raw.pack_id !== 'string' || raw.pack_id.length === 0) {
      return undefined;
    }
    const pack: Pack = new Pack();
    pack.id = raw.pack_id;
    pack.name = typeof raw.name === 'string' ? raw.name : raw.pack_id;
    pack.labelZh = typeof raw.labelZh === 'string' ? raw.labelZh : '';
    pack.source = 'builtin';
    pack.scene = BuiltinPackLoader.parseScene(raw.scene);
    pack.words = BuiltinPackLoader.parseWords(raw.words);
    return pack;
  }

  private static parseScene(raw: RawScene | undefined): SceneMetadata {
    const scene: SceneMetadata = new SceneMetadata();
    if (raw === undefined) {
      return scene;
    }
    if (typeof raw.bgPrimary === 'string') scene.bgPrimary = raw.bgPrimary;
    if (typeof raw.bgAccent === 'string') scene.bgAccent = raw.bgAccent;
    if (typeof raw.bossName === 'string') scene.bossName = raw.bossName;
    if (Array.isArray(raw.bossCandidates)) {
      scene.bossCandidates = raw.bossCandidates.filter(
        (n: number): boolean => typeof n === 'number'
      );
    }
    if (typeof raw.storyZh === 'string' && raw.storyZh.length > 0) {
      scene.storyZh = raw.storyZh;
    }
    if (Array.isArray(raw.monsterPlan)) {
      const plan: MonsterPlan = new MonsterPlan();
      const slots: MonsterSlot[] = [];
      for (let i = 0; i < raw.monsterPlan.length; i++) {
        const slot: MonsterSlot = new MonsterSlot();
        const r: RawSlot = raw.monsterPlan[i];
        slot.kind = BuiltinPackLoader.parseKind(r.kind);
        slot.catalogIndex = typeof r.catalogIndex === 'number'
          ? r.catalogIndex
          : monsterIndexForKind(slot.kind);
        slots.push(slot);
      }
      plan.slots = slots;
      scene.monsterPlan = plan;
    }
    return scene;
  }

  private static parseKind(s: string | undefined): MonsterKind {
    if (s === 'spelling') return MonsterKind.Spelling;
    if (s === 'review') return MonsterKind.Review;
    if (s === 'elite') return MonsterKind.Elite;
    if (s === 'boss') return MonsterKind.Boss;
    return MonsterKind.Normal;
  }

  private static parseWords(raw: RawWord[] | undefined): WordEntry[] {
    if (!Array.isArray(raw)) return [];
    const out: WordEntry[] = [];
    const seen: Set<string> = new Set<string>();
    for (let i = 0; i < raw.length; i++) {
      const r: RawWord = raw[i];
      const e: WordEntry = new WordEntry();
      e.id = r.id ?? '';
      e.word = r.word ?? '';
      e.meaningZh = r.meaningZh ?? '';
      e.category = r.category ?? '';
      e.difficulty = typeof r.difficulty === 'number' ? r.difficulty : 1;
      if (Array.isArray(r.distractors)) {
        const cleaned: string[] = [];
        for (let j = 0; j < r.distractors.length; j++) {
          const d: string = r.distractors[j];
          if (typeof d === 'string' && d.length > 0) cleaned.push(d);
        }
        if (cleaned.length > 0) e.distractors = cleaned;
      }
      if (r.example !== undefined && r.example !== null) {
        const ex: ExampleSentence = new ExampleSentence();
        ex.en = typeof r.example.en === 'string' ? r.example.en : '';
        ex.zh = typeof r.example.zh === 'string' ? r.example.zh : '';
        if (ex.en.length > 0 || ex.zh.length > 0) e.example = ex;
      }
      if (typeof r.illustrationUrl === 'string') e.illustrationUrl = r.illustrationUrl;
      if (typeof r.audioUrl === 'string') e.audioUrl = r.audioUrl;
      if (!e.isValid() || seen.has(e.id)) continue;
      seen.add(e.id);
      out.push(e);
    }
    return out;
  }

  /** Read all 5 builtin rawfiles and return them in a deterministic order. */
  static async loadAll(ctx: common.UIAbilityContext): Promise<Pack[]> {
    const packs: Pack[] = [];
    const decoder: util.TextDecoder = util.TextDecoder.create('utf-8');
    for (let i = 0; i < BUILTIN_PACK_FILES.length; i++) {
      const path: string = BUILTIN_PACK_FILES[i];
      try {
        const bytes: Uint8Array = await ctx.resourceManager.getRawFileContent(path);
        const text: string = decoder.decodeToString(bytes);
        const pack: Pack | undefined = BuiltinPackLoader.parsePackJson(text);
        if (pack !== undefined) packs.push(pack);
      } catch (err) {
        const be: BusinessError = err as BusinessError;
        console.error(`BuiltinPackLoader: ${path} failed: ${JSON.stringify(be)}`);
      }
    }
    return packs;
  }
}
```

```typescript
import { common } from '@kit.AbilityKit';
import { BusinessError } from '@ohos.base';

import { ParentFetchAdapter, ParentFetchResult } from './ParentApiClient';
import { StringPreferencesLike } from './WrongAnswerStore';
import { Pack, SceneMetadata } from '../models/Pack';
import { WordEntry, ExampleSentence } from '../models/WordEntry';
import { MonsterPlan, MonsterSlot } from '../models/AdventureRegion';
import { MonsterKind, monsterIndexForKind } from '../data/MonsterCatalog';

const KEY_BLOB: string = 'global_packs_v1';
const HEADER_IF_NONE_MATCH: string = 'If-None-Match';
const HEADER_ETAG: string = 'ETag';
const HEADER_ETAG_LOWER: string = 'etag';

class CachedEnvelope {
  etag: string = '';
  body: string = '';
  fetchedAtMs: number = 0;
}

interface RawEnvelope {
  etag?: string;
  body?: string;
  fetchedAtMs?: number;
}

interface RawSlot { kind?: string; catalogIndex?: number; }
interface RawScene {
  bgPrimary?: string; bgAccent?: string; bossName?: string;
  bossCandidates?: number[]; monsterPlan?: RawSlot[]; storyZh?: string;
}
interface RawWord {
  id?: string; word?: string; meaningZh?: string; meaning_zh?: string;
  category?: string; difficulty?: number; distractors?: string[];
  example?: { en?: string; zh?: string };
  illustrationUrl?: string; illustration_url?: string;
  audioUrl?: string; audio_url?: string;
}
interface RawPack {
  pack_id?: string; name?: string; labelZh?: string;
  version?: number; schema_version?: number; published_at?: string;
  scene?: RawScene; words?: RawWord[];
}
interface RawMerged {
  schema_version?: number;
  merged_at?: string;
  packs?: RawPack[];

export class GlobalPackEnsureResult {
  status: 'updated' | 'not_modified' | 'cleared' | 'cached_fallback' | 'no_data' = 'no_data';
  packs: Pack[] = [];
}

class RealGlobalPackPrefs implements StringPreferencesLike {
  private readonly handle: preferences.Preferences;
  constructor(handle: preferences.Preferences) { this.handle = handle; }
  async getString(key: string): Promise<string> {
    try {
      const v: preferences.ValueType = await this.handle.get(key, '');
      return typeof v === 'string' ? v : '';
    } catch (err) {
      console.error(`GlobalPackService.getString failed: ${JSON.stringify(err as BusinessError)}`);
      return '';
    }
  }
  async putString(key: string, value: string): Promise<void> {
    try {
      await this.handle.put(key, value);
    } catch (err) {
      console.error(`GlobalPackService.putString failed: ${JSON.stringify(err as BusinessError)}`);
    }
  }
  async flush(): Promise<void> {
    try {
      await this.handle.flush();
    } catch (err) {
      console.error(`GlobalPackService.flush failed: ${JSON.stringify(err as BusinessError)}`);
    }
  }

export class GlobalPackService {
  private readonly adapter: ParentFetchAdapter;
  private readonly baseUrl: string;
  private prefs?: StringPreferencesLike;
  private memoryEnv: CachedEnvelope | null = null;
  private memoryPacks: Pack[] = [];
  private nowMs: () => number = (): number => Date.now();

  constructor(adapter: ParentFetchAdapter, baseUrl: string) {
    this.adapter = adapter;
    this.baseUrl = GlobalPackService.trimTrailingSlash(baseUrl);
  }

  injectPreferences(p: StringPreferencesLike): void { this.prefs = p; }
  injectClock(now: () => number): void { this.nowMs = now; }

  async init(ctx: common.UIAbilityContext): Promise<void> {
    if (this.prefs !== undefined) return;
    try {
      const real: preferences.Preferences = await preferences.getPreferences(ctx, PREFS_NAME);
      this.prefs = new RealGlobalPackPrefs(real);
    } catch (err) {
      console.error(`GlobalPackService.init failed: ${JSON.stringify(err as BusinessError)}`);
    }
  }

  currentPacks(): Pack[] { return this.memoryPacks.slice(); }

  async warmFromCache(): Promise<Pack[]> {
    const env: CachedEnvelope | null = await this.readCache();
    if (env === null) {
      this.memoryEnv = null;
      this.memoryPacks = [];
      return [];
    }
    this.memoryEnv = env;
    this.memoryPacks = GlobalPackService.parseMergedBody(env.body);
    return this.memoryPacks;
  }

  async ensureLatest(): Promise<GlobalPackEnsureResult> {
    const out: GlobalPackEnsureResult = new GlobalPackEnsureResult();
    const url: string = `${this.baseUrl}/api/v1/public/global-packs/latest.json`;
    const headers: Record<string, string> = {};
    const cached: CachedEnvelope | null = this.memoryEnv ?? await this.readCache();
    if (cached !== null && cached.etag.length > 0) {
      headers[HEADER_IF_NONE_MATCH] = cached.etag;
    }

    let res: ParentFetchResult;
    try {
      res = await this.adapter.get(url, headers);
    } catch (_err) {
      out.status = 'cached_fallback';
      out.packs = this.memoryPacks;
      return out;
    }

    if (res.status === 304) {
      out.status = 'not_modified';
      out.packs = cached !== null ? GlobalPackService.parseMergedBody(cached.body) : this.memoryPacks;
      this.memoryPacks = out.packs;
      return out;
    }
    if (res.status === 204) {
      await this.clearCache();
      this.memoryEnv = null;
      this.memoryPacks = [];
      out.status = 'cleared';
      return out;
    }
    if (res.status === 200 || res.status === 201) {
      const etagHeader: string = this.adapter.lastResponseHeader(HEADER_ETAG);
      const etagFallback: string = this.adapter.lastResponseHeader(HEADER_ETAG_LOWER);
      const etag: string = etagHeader.length > 0 ? etagHeader : etagFallback;
      const env: CachedEnvelope = new CachedEnvelope();
      env.etag = etag; env.body = res.body; env.fetchedAtMs = this.nowMs();
      await this.writeCache(env);
      this.memoryEnv = env;
      this.memoryPacks = GlobalPackService.parseMergedBody(env.body);
      out.status = 'updated';
      out.packs = this.memoryPacks;
      return out;
    }
    out.status = 'cached_fallback';
    out.packs = this.memoryPacks;
    return out;
  }

  private async readCache(): Promise<CachedEnvelope | null> {
    if (this.prefs === undefined) return null;
    const raw: string = await this.prefs.getString(KEY_BLOB);
    if (raw.length === 0) return null;
    try {
      const env: RawEnvelope = JSON.parse(raw) as RawEnvelope;
      const out: CachedEnvelope = new CachedEnvelope();
      out.etag = typeof env.etag === 'string' ? env.etag : '';
      out.body = typeof env.body === 'string' ? env.body : '';
      out.fetchedAtMs = typeof env.fetchedAtMs === 'number' ? env.fetchedAtMs : 0;
      return out;
    } catch (_err) {
      return null;
    }
  }

  private async writeCache(env: CachedEnvelope): Promise<void> {
    if (this.prefs === undefined) return;
    const wire: RawEnvelope = { etag: env.etag, body: env.body, fetchedAtMs: env.fetchedAtMs };
    await this.prefs.putString(KEY_BLOB, JSON.stringify(wire));
    await this.prefs.flush();
  }

  private async clearCache(): Promise<void> {
    if (this.prefs === undefined) return;
    await this.prefs.putString(KEY_BLOB, '');
    await this.prefs.flush();
  }

  private static trimTrailingSlash(p: string): string {
    let end: number = p.length;
    while (end > 0 && p.charAt(end - 1) === '/') end -= 1;
    return p.substring(0, end);
  }

  static parseMergedBody(body: string): Pack[] {
    if (body.length === 0) return [];
    let raw: RawMerged;
    try {
      raw = JSON.parse(body) as RawMerged;
    } catch (_err) {
      return [];
    }
    if (raw === null || raw === undefined || !Array.isArray(raw.packs)) return [];
    const out: Pack[] = [];
    for (let i = 0; i < raw.packs.length; i++) {
      const pack: Pack | undefined = GlobalPackService.parseRawPack(raw.packs[i]);
      if (pack !== undefined) out.push(pack);
    }
    return out;
  }

  private static parseRawPack(r: RawPack): Pack | undefined {
    if (typeof r.pack_id !== 'string' || r.pack_id.length === 0) return undefined;
    const p: Pack = new Pack();
    p.id = r.pack_id;
    p.name = typeof r.name === 'string' ? r.name : r.pack_id;
    p.labelZh = typeof r.labelZh === 'string' ? r.labelZh : '';
    p.source = 'global';
    p.version = typeof r.version === 'number' ? r.version : 0;
    p.publishedAtMs = GlobalPackService.parseIsoMs(r.published_at);
    p.scene = GlobalPackService.parseScene(r.scene);
    p.words = GlobalPackService.parseWords(r.words);
    return p;
  }

  private static parseIsoMs(s: string | undefined): number {
    if (typeof s !== 'string' || s.length === 0) return 0;
    const ms: number = Date.parse(s);
    return Number.isNaN(ms) ? 0 : ms;
  }

  private static parseScene(raw: RawScene | undefined): SceneMetadata {
    const scene: SceneMetadata = new SceneMetadata();
    if (raw === undefined) return scene;
    if (typeof raw.bgPrimary === 'string') scene.bgPrimary = raw.bgPrimary;
    if (typeof raw.bgAccent === 'string') scene.bgAccent = raw.bgAccent;
    if (typeof raw.bossName === 'string') scene.bossName = raw.bossName;
    if (Array.isArray(raw.bossCandidates)) {
      scene.bossCandidates = raw.bossCandidates.filter((n: number): boolean => typeof n === 'number');
    }
    if (typeof raw.storyZh === 'string' && raw.storyZh.length > 0) scene.storyZh = raw.storyZh;
    if (Array.isArray(raw.monsterPlan)) {
      const plan: MonsterPlan = new MonsterPlan();
      const slots: MonsterSlot[] = [];
      for (let i = 0; i < raw.monsterPlan.length; i++) {
        const slot: MonsterSlot = new MonsterSlot();
        const r: RawSlot = raw.monsterPlan[i];
        slot.kind = GlobalPackService.parseKind(r.kind);
        slot.catalogIndex = typeof r.catalogIndex === 'number'
          ? r.catalogIndex : monsterIndexForKind(slot.kind);
        slots.push(slot);
      }
      plan.slots = slots;
      scene.monsterPlan = plan;
    }
    return scene;
  }

  private static parseKind(s: string | undefined): MonsterKind {
    if (s === 'spelling') return MonsterKind.Spelling;
    if (s === 'review') return MonsterKind.Review;
    if (s === 'elite') return MonsterKind.Elite;
    if (s === 'boss') return MonsterKind.Boss;
    return MonsterKind.Normal;
  }

  private static parseWords(raw: RawWord[] | undefined): WordEntry[] {
    if (!Array.isArray(raw)) return [];
    const out: WordEntry[] = [];
    const seen: Set<string> = new Set<string>();
    for (let i = 0; i < raw.length; i++) {
      const r: RawWord = raw[i];
      const e: WordEntry = new WordEntry();
      e.id = r.id ?? '';
      e.word = r.word ?? '';
      e.meaningZh = (typeof r.meaningZh === 'string' && r.meaningZh.length > 0) ? r.meaningZh
        : (typeof r.meaning_zh === 'string' ? r.meaning_zh : '');
      e.category = r.category ?? '';
      e.difficulty = typeof r.difficulty === 'number' ? r.difficulty : 1;
      if (Array.isArray(r.distractors)) {
        const cleaned: string[] = [];
        for (let j = 0; j < r.distractors.length; j++) {
          const d: string = r.distractors[j];
          if (typeof d === 'string' && d.length > 0) cleaned.push(d);
        }
        if (cleaned.length > 0) e.distractors = cleaned;
      }
      if (r.example !== undefined && r.example !== null) {
        const ex: ExampleSentence = new ExampleSentence();
        ex.en = typeof r.example.en === 'string' ? r.example.en : '';
        ex.zh = typeof r.example.zh === 'string' ? r.example.zh : '';
        if (ex.en.length > 0 || ex.zh.length > 0) e.example = ex;
      }
      const illust: string | undefined = (typeof r.illustrationUrl === 'string' && r.illustrationUrl.length > 0)
        ? r.illustrationUrl
        : (typeof r.illustration_url === 'string' && r.illustration_url.length > 0 ? r.illustration_url : undefined);
      if (illust !== undefined) e.illustrationUrl = illust;
      const audio: string | undefined = (typeof r.audioUrl === 'string' && r.audioUrl.length > 0)
        ? r.audioUrl
        : (typeof r.audio_url === 'string' && r.audio_url.length > 0 ? r.audio_url : undefined);
      if (audio !== undefined) e.audioUrl = audio;
      if (!e.isValid() || seen.has(e.id)) continue;
      seen.add(e.id);
      out.push(e);
    }
    return out;
  }
```

```typescript
// entry/src/main/ets/services/PackLibrary.ets
import { Pack, SceneMetadata } from '../models/Pack';

const FALLBACK_PALETTE: SceneMetadata[] = ((): SceneMetadata[] => {
  const a: SceneMetadata = new SceneMetadata();
  a.bgPrimary = '#FFF6E0'; a.bgAccent = '#FFD49A'; a.bossName = 'Orchard Sentinel';
  const b: SceneMetadata = new SceneMetadata();
  b.bgPrimary = '#E8F0FE'; b.bgAccent = '#AECBFA'; b.bossName = 'Clock Wizard';
  const c: SceneMetadata = new SceneMetadata();
  c.bgPrimary = '#FFF1E6'; c.bgAccent = '#F4B98A'; c.bossName = 'Toy Knight';
  return [a, b, c];
})();

/**
 * Hash a string to a non-negative 32-bit int via djb2. Used to stably
 * pick a fallback palette index for family packs.
 */
function djb2Hash(s: string): number {
  let h: number = 5381;
  for (let i: number = 0; i < s.length; i++) {
    h = ((h << 5) - h) + s.charCodeAt(i);
    h = h | 0;
  }
  return h < 0 ? -h : h;
}

function isEmptyScene(s: SceneMetadata): boolean {
  return s.bgPrimary === '#FFFFFF' && s.bgAccent === '#FFFFFF' && s.bossName === '';
}

/**
 * V0.6.5 — composes the three pack source layers into a single
 * `Pack[]`. Override priority: family > global > builtin on same id.
 * Scene fallback (per spec §5.4):
 *   1. own non-empty scene -> keep
 *   2. else look up same-id builtin's scene
 *   3. else hash(id) % FALLBACK_PALETTE.length
 */
export class PackLibrary {
  private builtins: Pack[] = [];
  private globals: Pack[] = [];
  private families: Pack[] = [];

  setBuiltins(packs: Pack[]): void { this.builtins = packs.slice(); }
  setGlobals(packs: Pack[]): void { this.globals = packs.slice(); }
  setFamilies(packs: Pack[]): void { this.families = packs.slice(); }

  /** Union the three layers, family-priority, with scene fallback. */
  allPacks(): Pack[] {
    const builtinById: Map<string, Pack> = new Map();
    for (let i = 0; i < this.builtins.length; i++) {
      builtinById.set(this.builtins[i].id, this.builtins[i]);
    }
    const out: Map<string, Pack> = new Map();
    for (let i = 0; i < this.builtins.length; i++) {
      out.set(this.builtins[i].id, this.builtins[i]);
    }
    for (let i = 0; i < this.globals.length; i++) {
      const g: Pack = this.globals[i];
      const merged: Pack = PackLibrary.applySceneFallback(g, builtinById);
      out.set(g.id, merged);
    }
    for (let i = 0; i < this.families.length; i++) {
      const f: Pack = this.families[i];
      const merged: Pack = PackLibrary.applySceneFallback(f, builtinById);
      out.set(f.id, merged);
    }
    const list: Pack[] = [];
    out.forEach((p: Pack): void => { list.push(p); });
    return list;
  }

  byId(id: string): Pack | undefined {
    const list: Pack[] = this.allPacks();
    for (let i = 0; i < list.length; i++) {
      if (list[i].id === id) return list[i];
    }
    return undefined;
  }

  private static applySceneFallback(p: Pack, builtinById: Map<string, Pack>): Pack {
    if (!isEmptyScene(p.scene)) return p;
    const sib: Pack | undefined = builtinById.get(p.id);
    if (sib !== undefined && !isEmptyScene(sib.scene)) {
      const merged: Pack = new Pack();
      merged.id = p.id; merged.name = p.name; merged.labelZh = p.labelZh;
      merged.source = p.source; merged.version = p.version;
      merged.publishedAtMs = p.publishedAtMs;
      merged.scene = sib.scene;
      merged.words = p.words.slice();
      return merged;
    }
    const idx: number = djb2Hash(p.id) % FALLBACK_PALETTE.length;
    const merged: Pack = new Pack();
    merged.id = p.id; merged.name = p.name; merged.labelZh = p.labelZh;
    merged.source = p.source; merged.version = p.version;
    merged.publishedAtMs = p.publishedAtMs;
    merged.scene = FALLBACK_PALETTE[idx];
    merged.words = p.words.slice();
    return merged;
  }
}
```

```typescript
export const MAX_ACTIVE_PACKS: number = 5;
export const PERFECT_THRESHOLD: number = 3;

interface RawSelection {
  schemaVersion?: number;
  activePackIds?: string[];
  pinnedPackIds?: string[];

  private readonly handle: preferences.Preferences;
  constructor(handle: preferences.Preferences) { this.handle = handle; }
  async getString(k: string): Promise<string> {
    try {
      const v: preferences.ValueType = await this.handle.get(k, '');
      return typeof v === 'string' ? v : '';
    } catch (err) {
      console.error(`PackSelectionService.getString: ${JSON.stringify(err as BusinessError)}`);
      return '';
    }
  }
  async putString(k: string, v: string): Promise<void> {
    try { await this.handle.put(k, v); } catch (err) {
      console.error(`PackSelectionService.putString: ${JSON.stringify(err as BusinessError)}`);
    }
  }
  async flush(): Promise<void> {
    try { await this.handle.flush(); } catch (err) {
      console.error(`PackSelectionService.flush: ${JSON.stringify(err as BusinessError)}`);
    }
  }
}

/**
 * V0.6.5 — owns the device-local active-pack selection, pin state, and
 * perfect-adventure counters. Pure-state helpers + minimal persistence;
 * auto-rotation lives in `recordPerfectAdventure` (Task 18).
 */
export class PackSelectionService {
  private prefs?: StringPreferencesLike;
  private activeIds: string[] = [];
  private pinnedIds: Set<string> = new Set<string>();
  private perfectScores: Map<string, number> = new Map<string, number>();

  injectPreferences(p: StringPreferencesLike): void { this.prefs = p; }

  async init(ctx: common.UIAbilityContext, defaults: string[]): Promise<void> {
    if (this.prefs === undefined) {
      try {
        const real: preferences.Preferences = await preferences.getPreferences(ctx, PREFS_NAME);
        this.prefs = new RealPrefs(real);
      } catch (err) {
        console.error(`PackSelectionService.init: ${JSON.stringify(err as BusinessError)}`);
      }
    }
    await this.initWithDefaults(defaults);
  }

  async initWithDefaults(defaults: string[]): Promise<void> {
    const raw: string = this.prefs !== undefined ? await this.prefs.getString(KEY_BLOB) : '';
    if (raw.length === 0) {
      this.activeIds = defaults.slice(0, MAX_ACTIVE_PACKS);
      this.pinnedIds = new Set<string>();
      this.perfectScores = new Map<string, number>();
      await this.persist();
      return;
    }
    try {
      const parsed: RawSelection = JSON.parse(raw) as RawSelection;
      this.activeIds = Array.isArray(parsed.activePackIds) ? parsed.activePackIds.slice(0, MAX_ACTIVE_PACKS) : defaults.slice(0, MAX_ACTIVE_PACKS);
      this.pinnedIds = new Set<string>(Array.isArray(parsed.pinnedPackIds) ? parsed.pinnedPackIds : []);
      this.perfectScores = new Map<string, number>();
      const counts = parsed.perfectScoresByPack ?? {};
      Object.keys(counts).forEach((k: string): void => {
        const v: number = counts[k];
        if (typeof v === 'number') this.perfectScores.set(k, v);
      });
    } catch (_err) {
      this.activeIds = defaults.slice(0, MAX_ACTIVE_PACKS);
      this.pinnedIds = new Set<string>();
      this.perfectScores = new Map<string, number>();
      await this.persist();
    }
  }

  getActiveIds(): string[] { return this.activeIds.slice(); }

  async setActiveIds(ids: string[]): Promise<void> {
    if (ids.length > MAX_ACTIVE_PACKS) {
      throw new Error(`active set exceeds ${MAX_ACTIVE_PACKS}`);
    }
    const seen: Set<string> = new Set();
    for (let i = 0; i < ids.length; i++) {
      if (seen.has(ids[i])) throw new Error(`duplicate id: ${ids[i]}`);
      seen.add(ids[i]);
    }
    this.activeIds = ids.slice();
    // Drop pinned ids no longer in the active set.
    const next: Set<string> = new Set<string>();
    this.pinnedIds.forEach((id: string): void => {
      if (seen.has(id)) next.add(id);
    });
    this.pinnedIds = next;
    await this.persist();
  }

  isPinned(id: string): boolean { return this.pinnedIds.has(id); }
  pinnedIdsList(): string[] { const out: string[] = []; this.pinnedIds.forEach((id: string): void => { out.push(id); }); return out; }

  async togglePin(id: string): Promise<void> {
    if (this.activeIds.indexOf(id) < 0) return;  // pin only matters for active packs
    if (this.pinnedIds.has(id)) this.pinnedIds.delete(id);
    else this.pinnedIds.add(id);
    await this.persist();
  }

  perfectScore(id: string): number { return this.perfectScores.get(id) ?? 0; }

  /**
   * Drop ids not present in `availableIds` from active + pinned and
   * persist. Called after PackLibrary.refresh() so a deleted family pack
   * shrinks the active row immediately.
   */
  pruneAgainstLibrary(availableIds: Set<string>): void {
    this.activeIds = this.activeIds.filter((id: string): boolean => availableIds.has(id));
    const next: Set<string> = new Set<string>();
    this.pinnedIds.forEach((id: string): void => {
      if (availableIds.has(id)) next.add(id);
    });
    this.pinnedIds = next;
    // Persist asynchronously; callers that need durability await persist().
    this.persist().catch((_err: BusinessError): void => {});
  }

  async persist(): Promise<void> {
    if (this.prefs === undefined) return;
    const counts: Record<string, number> = {};
    this.perfectScores.forEach((v: number, k: string): void => { counts[k] = v; });
    const wire: RawSelection = {
      schemaVersion: 1,
      activePackIds: this.activeIds.slice(),
      pinnedPackIds: this.pinnedIdsList(),
      perfectScoresByPack: counts,
    };
    await this.prefs.putString(KEY_BLOB, JSON.stringify(wire));
    await this.prefs.flush();
  }
}
```

```typescript
export interface PackCandidateProvider {
  /** Return next id to swap into active set, or undefined if pool is exhausted. */
  next(activeIds: Set<string>): string | undefined;
}

export class PerfectAdventureResult {
  rotated: boolean = false;
  swappedOutId: string = '';
  swappedInId: string = '';
}

// Append as a method of class PackSelectionService:
async recordPerfectAdventure(
  packId: string, provider: PackCandidateProvider
): Promise<PerfectAdventureResult> {
  const out: PerfectAdventureResult = new PerfectAdventureResult();
  const cur: number = this.perfectScores.get(packId) ?? 0;
  this.perfectScores.set(packId, cur + 1);

  if (this.activeIds.indexOf(packId) < 0) {
    await this.persist();
    return out;
  }
  if (this.pinnedIds.has(packId)) {
    await this.persist();
    return out;
  }
  if (cur + 1 < PERFECT_THRESHOLD) {
    await this.persist();
    return out;
  }

  const activeSet: Set<string> = new Set(this.activeIds);
  const candidateId: string | undefined = provider.next(activeSet);
  if (candidateId === undefined) {
    // Pool exhausted; keep the count, freeze active set.
    await this.persist();
    return out;
  }
  // Rotate.
  this.activeIds = this.activeIds.filter((id: string): boolean => id !== packId);
  this.activeIds.push(candidateId);
  this.perfectScores.set(packId, 0);
  out.rotated = true;
  out.swappedOutId = packId;
  out.swappedInId = candidateId;
  await this.persist();
  return out;
}
```

```typescript
 * V0.6.5 — assembles the full PackLibrary at app start.
 * Built-in packs come from rawfile; global from the cached
 * `wordmagic_global_packs` slot; family from `wordmagic_family_packs`.
 *
 * `bootstrap()` is read-only — it never touches the network. Network
 * sync happens only when ConfigPage's "同步全部词包" button explicitly
 * calls into GlobalPackService / FamilyPackService.
 */
export class WordPackBootstrapper {
  static async bootstrap(ctx: common.UIAbilityContext): Promise<PackLibrary> {
    const lib: PackLibrary = new PackLibrary();
    const builtinPromise: Promise<Pack[]> = BuiltinPackLoader.loadAll(ctx);

    const baseUrl: string = backendBaseUrl();
    const adapter = ParentApiClient.defaultAdapter();
    const gps: GlobalPackService = new GlobalPackService(adapter, baseUrl);
    await gps.init(ctx);
    const globalPromise: Promise<Pack[]> = gps.warmFromCache();

    const fps: FamilyPackService = new FamilyPackService(
      // FamilyPackService keeps its existing adapter contract; head() is
      // used by the banner-detection path. ParentApiClient.defaultAdapter()
      // already supplies head().
      adapter as never, baseUrl);
    await fps.init(ctx);
    const familyPromise: Promise<Pack[]> = WordPackBootstrapper.familyPacksFromService(fps);

    const builtins: Pack[] = await builtinPromise;
    const globals: Pack[] = await globalPromise;
    const families: Pack[] = await familyPromise;
    lib.setBuiltins(builtins);
    lib.setGlobals(globals);
    lib.setFamilies(families);
    return lib;
  }

  /** Adapt the existing FamilyPackService blob shape into Pack[]. */
  private static async familyPacksFromService(fps: FamilyPackService): Promise<Pack[]> {
    const blob = await fps.warmFromCache();
    if (blob === null) return [];
    // FamilyPacksBlob.packs[i].words is `object[]` (RawWord). Reuse the
    // global parser by re-serialising, since the wire shapes match
    // closely; this avoids duplicating the field-by-field plumbing.
    const synthetic = {
      schema_version: blob.schemaVersion,
      merged_at: '',
      packs: blob.packs.map(p => ({
        pack_id: p.packId,
        name: p.name,
        version: p.version,
        schema_version: p.schemaVersion,
        published_at: '',
        words: p.words,
      })),
    };
    const parsed: Pack[] = GlobalPackService.parseMergedBody(JSON.stringify(synthetic));
    // Mark all packs as 'family' source (the parser would mark them 'global').
    return parsed.map((pk: Pack): Pack => {
      const f: Pack = pk; f.source = 'family'; return f;
    });
  }
```

```typescript
private buildCandidateProvider(): PackCandidateProvider {
  const lib: PackLibrary | null = this.library;
  return {
    next: (active: Set<string>): string | undefined => {
      if (lib === null) return undefined;
      const all: Pack[] = lib.allPacks();
      const ranked: Pack[] = all.filter((p: Pack): boolean => !active.has(p.id));
      ranked.sort((a: Pack, b: Pack): number => {
        const sa: number = a.source === 'family' ? 0 : (a.source === 'global' ? 1 : 2);
        const sb: number = b.source === 'family' ? 0 : (b.source === 'global' ? 1 : 2);
        if (sa !== sb) return sa - sb;
        if (a.publishedAtMs !== b.publishedAtMs) return b.publishedAtMs - a.publishedAtMs;
        if (a.id < b.id) return -1;
        if (a.id > b.id) return 1;
        return 0;
      });
      return ranked.length > 0 ? ranked[0].id : undefined;
    },
  };
}
```

## Task 27: Legacy route aliases — surface new prefixes on existing handlers

**Files:**

Per [.cursor/rules/api-route-pattern.mdc](.cursor/rules/api-route-pattern.mdc) the project-wide URL convention is `/api/v1/admin/**` + `/api/v1/public/**` + `/api/v1/family/{family_id}/**`. Many existing routers don't match. To roll the convention out without breaking shipped v0.6.x clients, this task **adds aliases**: each legacy URL gains a sibling at the new-pattern path that routes to the **same handler function**, with `include_in_schema=False` so /docs only shows the new shape going forward.

> **Forward-compatibility contract:** legacy URLs keep returning their existing responses unchanged. Removal of the legacy URLs (and corresponding handler relocations) is scheduled for v0.6.6+ once every shipped HarmonyOS build has been seen migrating to the new URLs.

### Alias map

| Legacy URL                                            | New URL                                                            | Notes                                                                      |
| ----------------------------------------------------- | ------------------------------------------------------------------ | -------------------------------------------------------------------------- |
| `GET /api/v1/health`                                  | `GET /api/v1/public/health`                                        | anonymous probe                                                            |
| `GET /api/v1/packs/latest.json`                       | `GET /api/v1/public/packs/latest.json`                             | legacy mega-pack; new clients consume `/public/global-packs/latest.json` instead |
| `POST /api/v1/auth/login`, `GET /api/v1/auth/me`      | `POST /api/v1/admin/auth/login`, `GET /api/v1/admin/auth/me`       | admin auth                                                                 |
| `…/api/v1/parent/<rest>`                              | `…/api/v1/family/{family_id}/<rest>`                                | every parent-side router (parent_auth, parent_family_pack, parent_inbox, parent_api, parent_account) |
| `…/api/v1/child/<rest>`                               | `…/api/v1/family/{family_id}/<rest>`                                | every child-device router (child_word_stats, child_family_pack, child_wishlist) |
| `/parent/<rest>` (HTML)                               | `/family/{family_id}/<rest>` (HTML)                                | parent_pages, parent_inbox html, parent_account html                       |

`{family_id}` in the new URLs is a **decorative path parameter for v0.6.5**: the underlying handlers continue to read `family_id` from the existing parent session / Bearer device-token. v0.6.6 will land "path param must match session family_id" enforcement; v0.6.5 explicitly accepts any string and the test suite documents the expectation.

### Pre-login carve-out

Endpoints that fire before any session exists (e.g. parent OTP exchange, future SMS callback) cannot live under `/api/v1/family/{family_id}/...` because there is no `family_id` to put in the URL. Per the rule's "What about pre-login endpoints?" clause they belong under `/api/v1/public/auth/...`. **In v0.6.5 we do NOT alias these** — `/api/v1/parent/auth/login` keeps its current path and is already callable from any unauthenticated client. The only auth alias added by this task is the **admin** login (`/api/v1/auth/login` → `/api/v1/admin/auth/login`), which fits the rule cleanly.

```python
# server/app/routers/legacy_route_aliases.py
"""V0.6.5 — Map existing legacy URLs onto the new prefix convention.

Per .cursor/rules/api-route-pattern.mdc, all new HTTP surfaces live under
`/api/v1/admin/**`, `/api/v1/public/**`, or `/api/v1/family/{family_id}/**`.
This module adds **path aliases** pointing to the SAME handler functions
already registered by the legacy routers, so v0.6.5 clients can adopt the
new URL shape immediately while v0.6.4 clients keep working through the
legacy paths.

Aliases are excluded from the OpenAPI schema (`include_in_schema=False`)
so /docs only shows the new shape going forward; legacy URLs remain
visible in /docs during the transition window.

Removal of legacy URLs is scheduled for v0.6.6+ once every shipped
HarmonyOS build has been seen migrating to the new URLs.
"""

from __future__ import annotations

from fastapi import FastAPI
from starlette.routing import Route

# Exact-path rewrites (one-to-one).
_EXACT: dict[str, str] = {
    "/api/v1/health":                  "/api/v1/public/health",
    "/api/v1/packs/latest.json":       "/api/v1/public/packs/latest.json",
    "/api/v1/auth/login":              "/api/v1/admin/auth/login",
    "/api/v1/auth/me":                 "/api/v1/admin/auth/me",
}

# Prefix rewrites (apply to anything starting with `src` AS LONG AS the
# resulting path doesn't collide with `_EXACT`'s left-hand side).
# Order matters — first match wins.
_PREFIX: list[tuple[str, str]] = [
    ("/api/v1/parent/", "/api/v1/family/{family_id}/"),
    ("/api/v1/child/",  "/api/v1/family/{family_id}/"),
    ("/parent/",        "/family/{family_id}/"),
]


def _alias_for(path: str) -> str | None:
    if path in _EXACT:
        return _EXACT[path]
    for src, tgt in _PREFIX:
        if path.startswith(src):
            return tgt + path[len(src):]
    return None


def attach_legacy_aliases(app: FastAPI) -> int:
    """Register an alias Route for every legacy path. Returns the count
    of aliases added (handy for log lines / smoke tests).

    MUST be called AFTER every `app.include_router(...)` so this function
    can see all legacy routes already registered on `app.routes`.
    """
    new_routes: list[Route] = []
    for route in list(app.routes):
        if not isinstance(route, Route):
            continue
        new_path = _alias_for(route.path)
        if new_path is None:
            continue
        new_routes.append(
            Route(
                path=new_path,
                endpoint=route.endpoint,
                methods=route.methods,
                name=f"alias_{route.name}",
                include_in_schema=False,
            )
        )
    app.routes.extend(new_routes)
    return len(new_routes)
```
