# V0.5 Client Follow-up Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans (chosen by operator).
> Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Bring the HarmonyOS client up to par with the V0.5.7 server: stop the "Failed to load words" emulator regression, switch to operator-triggered cache sync, consume schema_v 2/4/5 fields end-to-end, add an in-app admin shell, and ship a deterministic e2e test fixture path.

**Architecture:** Six TDD-disciplined sub-versions A → F (one git commit each). Each sub-version is independently shippable. The first three lift the client to the new schema; D adds the lazy on-disk asset cache; E surfaces categories on HomePage; F adds a parent-PIN-gated admin shell behind a debug-only entry. Cold-start bootstrap is read-only after this branch — the only path that mutates the on-disk cache is the manual sync button on ConfigPage.

**Tech Stack:** ArkTS + HarmonyOS NEXT (DevEco-managed). `@ohos/hypium` for unit tests, `ohos.net.http` for HTTP, `ohos.data.preferences` for KV, `ohos.file.fs` for blob disk cache, `ohos.multimedia.media` for audio playback (already used). No new third-party dependencies.

```ts
import { common } from '@kit.AbilityKit';
import { WordEntry } from '../models/WordEntry';
import { WordRepository } from './WordRepository';
import { WordPackCache } from './WordPackCache';
import { RemoteWordPackService } from './RemoteWordPackService';
import { latestPackUrl } from './RemoteWordPackConfig';
import { MIN_REPO_SIZE } from './QuestionGenerator';

export type RawfileLoader = () => Promise<WordEntry[]>;

export interface BootstrapDeps {
  cache: WordPackCache;
  remote?: RemoteWordPackService;
  rawfileLoader: RawfileLoader;
  serverUrl: string;
}

/**
 * V0.5 client follow-up: cold-start orchestrator for the word pack.
 *
 * Read-only: bootstrap NEVER touches the network. The only path that
 * writes the cache is `WordPackSyncService.syncOnce`, triggered by an
 * operator tap on ConfigPage. Removing the background refresh kills
 * the cache-poisoning class of bug that surfaced in V0.5.7 (prod was
 * serving a 1-word pack; the old auto-refresh wrote it to disk; next
 * cold start handed a 1-word repo to QuestionGenerator and threw).
 *
 * Sync section (returns within tens of ms):
 *   1. read cache.
 *   2. if cache parses to >= MIN_REPO_SIZE valid words, build the
 *      WordRepository from it.
 *   3. otherwise (empty / corrupt / too small), fall back to the
 *      caller-provided rawfile loader.
 *
 * `BootstrapDeps.remote` stays in the dependency bag because manual
 * sync still needs it; bootstrap itself ignores the field.
 */
export class WordPackBootstrapper {
  static async forContext(ctx: common.UIAbilityContext): Promise<BootstrapDeps> {
    const cache: WordPackCache = new WordPackCache();
    await cache.init(ctx);
    const remote: RemoteWordPackService = new RemoteWordPackService();
    const repo: WordRepository = new WordRepository();
    const deps: BootstrapDeps = {
      cache,
      remote,
      rawfileLoader: async (): Promise<WordEntry[]> => {
        await repo.loadFromRawfile(ctx);
        return repo.all();
      },
      serverUrl: latestPackUrl(),
    };
    return deps;
  }

  async bootstrap(deps: BootstrapDeps): Promise<WordRepository> {
    const cached: string = await deps.cache.read();
    const repo: WordRepository = new WordRepository();
    if (cached.length > 0) {
      const entries: WordEntry[] = WordRepository.parseWordsJson(cached);
      if (entries.length >= MIN_REPO_SIZE) {
        repo.setEntries(entries);
        return repo;
      }
    }
    repo.setEntries(await deps.rawfileLoader());
    return repo;
  }
}
```

```jsonc
  "fetchedAt": "2026-05-01T08:42:00.000Z"
}
```

Backwards compatibility: if a cached value is a bare pack JSON (not an envelope), `WordPackCache.read()` still returns that string so VC0.5.A clients keep working until they sync once. After the first successful sync, the slot holds the envelope.

```ts
import { common } from '@kit.AbilityKit';
import { BusinessError } from '@ohos.base';
import { StringPreferencesLike } from './WrongAnswerStore';

const PREFS_NAME: string = 'word_pack_cache';
const PACK_KEY: string = 'pack_v2';

/**
 * Envelope written by WordPackSyncService after a successful pull.
 * Stored as a single JSON string in `pack_v2`. Older installs may
 * have a bare pack JSON in the same slot — `read()` keeps returning
 * just the body so the cold-start bootstrap is identical for both.
 */
export class WordPackCacheRecord {
  body: string = '';
  etag: string = '';
  schemaVersion: number = 1;
  fetchedAt: string = '';
}

class RealPrefs implements StringPreferencesLike {
  private readonly handle: preferences.Preferences;
  constructor(handle: preferences.Preferences) { this.handle = handle; }
  async getString(key: string): Promise<string> {
    const v: preferences.ValueType = await this.handle.get(key, '');
    return typeof v === 'string' ? v : '';
  }
  putString(key: string, value: string): Promise<void> {
    return this.handle.put(key, value);
  }
  flush(): Promise<void> {
    return this.handle.flush();
  }
}

export class WordPackCache {
  private prefs?: StringPreferencesLike;

  injectPreferencesForTest(p: StringPreferencesLike): void {
    this.prefs = p;
  }

  async init(ctx: common.UIAbilityContext): Promise<void> {
    if (this.prefs !== undefined) {
      return;
    }
    try {
      const handle: preferences.Preferences = await preferences.getPreferences(ctx, PREFS_NAME);
      this.prefs = new RealPrefs(handle);
    } catch (err) {
      const be: BusinessError = err as BusinessError;
      console.error(`WordPackCache.init failed: ${JSON.stringify(be)}`);
    }
  }

  /**
   * Returns just the pack body so that cold-start callers (the
   * bootstrapper) don't have to know about envelopes.
   */
  async read(): Promise<string> {
    if (this.prefs === undefined) {
      return '';
    }
    const raw: string = await this.prefs.getString(PACK_KEY);
    if (raw.length === 0) {
      return '';
    }
    const env: WordPackCacheRecord | undefined = WordPackCache.tryParseEnvelope(raw);
    if (env !== undefined) {
      return WordPackCache.looksValidBody(env.body) ? env.body : '';
    }
    return WordPackCache.looksValidBody(raw) ? raw : '';
  }

  async readRecord(): Promise<WordPackCacheRecord | undefined> {
    if (this.prefs === undefined) {
      return undefined;
    }
    const raw: string = await this.prefs.getString(PACK_KEY);
    if (raw.length === 0) {
      return undefined;
    }
    const env: WordPackCacheRecord | undefined = WordPackCache.tryParseEnvelope(raw);
    if (env === undefined) {
      return undefined;
    }
    if (!WordPackCache.looksValidBody(env.body)) {
      return undefined;
    }
    return env;
  }

  /**
   * Legacy bare-body write kept for callers that haven't been moved
   * to writeRecord (none today; both bootstrapper and sync path use
   * writeRecord). Validates body shape before persisting.
   */
  async write(packJson: string): Promise<void> {
    if (this.prefs === undefined) {
      return;
    }
    if (!WordPackCache.looksValidBody(packJson)) {
      return;
    }
    await this.prefs.putString(PACK_KEY, packJson);
    await this.prefs.flush();
  }

  async writeRecord(body: string, etag: string, schemaVersion: number): Promise<void> {
    if (this.prefs === undefined) {
      return;
    }
    if (!WordPackCache.looksValidBody(body)) {
      return;
    }
    const rec: WordPackCacheRecord = new WordPackCacheRecord();
    rec.body = body;
    rec.etag = etag;
    rec.schemaVersion = schemaVersion;
    rec.fetchedAt = new Date().toISOString();
    await this.prefs.putString(PACK_KEY, JSON.stringify(rec));
    await this.prefs.flush();
  }

  async touchFetchedAt(): Promise<void> {
    const rec: WordPackCacheRecord | undefined = await this.readRecord();
    if (rec === undefined || this.prefs === undefined) {
      return;
    }
    rec.fetchedAt = new Date().toISOString();
    await this.prefs.putString(PACK_KEY, JSON.stringify(rec));
    await this.prefs.flush();
  }

  async clearForTest(ctx: common.UIAbilityContext): Promise<void> {
    if (this.prefs === undefined) {
      await this.init(ctx);
    }
    if (this.prefs === undefined) {
      return;
    }
    await this.prefs.putString(PACK_KEY, '');
    await this.prefs.flush();
  }

  static looksValid(raw: string): boolean {
    if (raw.length === 0) {
      return false;
    }
    const env: WordPackCacheRecord | undefined = WordPackCache.tryParseEnvelope(raw);
    if (env !== undefined) {
      return WordPackCache.looksValidBody(env.body);
    }
    return WordPackCache.looksValidBody(raw);
  }

  private static tryParseEnvelope(raw: string): WordPackCacheRecord | undefined {
    let parsed: object;
    try {
      parsed = JSON.parse(raw) as object;
    } catch (_err) {
      return undefined;
    }
    if (parsed === null || parsed === undefined) {
      return undefined;
    }
    const obj: Record<string, Object> = parsed as Record<string, Object>;
    const body: Object | undefined = obj.body;
    if (typeof body !== 'string') {
      return undefined;
    }
    const rec: WordPackCacheRecord = new WordPackCacheRecord();
    rec.body = body;
    rec.etag = typeof obj.etag === 'string' ? (obj.etag as string) : '';
    rec.schemaVersion = typeof obj.schemaVersion === 'number'
      ? (obj.schemaVersion as number)
      : 1;
    rec.fetchedAt = typeof obj.fetchedAt === 'string' ? (obj.fetchedAt as string) : '';
    return rec;
  }

  private static looksValidBody(body: string): boolean {
    if (body.length === 0) {
      return false;
    }
    let parsed: object;
    try {
      parsed = JSON.parse(body) as object;
    } catch (_err) {
      return false;
    }
    if (parsed === null || parsed === undefined) {
      return false;
    }
    const obj: Record<string, Object> = parsed as Record<string, Object>;
    const words: Object | undefined = obj.words;
    if (words === undefined || words === null) {
      return false;
```

```ts
import { WordPackCache, WordPackCacheRecord } from './WordPackCache';
import { RemoteWordPackService, RemoteFetchResult } from './RemoteWordPackService';
import { latestPackUrl } from './RemoteWordPackConfig';

export enum SyncStatus {
  Updated = 'updated',
  UpToDate = 'up_to_date',
  NetworkError = 'network_error',
  HttpError = 'http_error',
  RejectedTooSmall = 'rejected_too_small',
}

export class SyncOutcome {
  status: SyncStatus = SyncStatus.NetworkError;
  cachedVersion: number = 0;
  cachedSchemaVersion: number = 0;
  httpStatus: number = 0;
}

const MIN_WORDS_PER_PACK: number = 3;

/**
 * V0.5 client follow-up: operator-triggered pack sync. The single
 * call site is the "同步词包" button on ConfigPage. There is no
 * background scheduler — the previous implementation auto-refreshed
 * on every cold start and ended up persisting whatever shape prod
 * happened to be in (including a 1-word pack).
 *
 * Returns a SyncOutcome the UI maps to a toast. The cache slot is
 * never mutated on a non-2xx response, on a network error, or on a
 * 200 that contains fewer than 3 valid words.
 */
export class WordPackSyncService {
  private readonly remote: RemoteWordPackService;
  private readonly cache: WordPackCache;
  private readonly url: string;

  constructor(remote: RemoteWordPackService, cache: WordPackCache, url?: string) {
    this.remote = remote;
    this.cache = cache;
    this.url = (url !== undefined && url.length > 0) ? url : latestPackUrl();
  }

  async syncOnce(): Promise<SyncOutcome> {
    const out: SyncOutcome = new SyncOutcome();
    const existing: WordPackCacheRecord | undefined = await this.cache.readRecord();
    const ifNoneMatch: string | undefined =
      existing !== undefined && existing.etag.length > 0 ? existing.etag : undefined;

    const res: RemoteFetchResult = await this.remote.fetchLatest(this.url, ifNoneMatch);
    if (res.status === 0) {
      out.status = SyncStatus.NetworkError;
      WordPackSyncService.fillCachedVersionInfo(out, existing);
      return out;
    }
    if (res.status === 304) {
      await this.cache.touchFetchedAt();
      out.status = SyncStatus.UpToDate;
      WordPackSyncService.fillCachedVersionInfo(out, existing);
      return out;
    }
    if (res.status !== 200) {
      out.status = SyncStatus.HttpError;
      out.httpStatus = res.status;
      WordPackSyncService.fillCachedVersionInfo(out, existing);
      return out;
    }

    const parsed: PackHeader | undefined = WordPackSyncService.tryReadHeader(res.body);
    if (parsed === undefined || parsed.wordsLength < MIN_WORDS_PER_PACK) {
      out.status = SyncStatus.RejectedTooSmall;
      WordPackSyncService.fillCachedVersionInfo(out, existing);
      return out;
    }
    await this.cache.writeRecord(res.body, res.etag, parsed.schemaVersion);
    out.status = SyncStatus.Updated;
    out.cachedVersion = parsed.version;
    out.cachedSchemaVersion = parsed.schemaVersion;
    return out;
  }

  private static fillCachedVersionInfo(out: SyncOutcome, existing?: WordPackCacheRecord): void {
    if (existing === undefined) {
      return;
    }
    const header: PackHeader | undefined = WordPackSyncService.tryReadHeader(existing.body);
    if (header !== undefined) {
      out.cachedVersion = header.version;
      out.cachedSchemaVersion = header.schemaVersion;
    }
  }

  private static tryReadHeader(body: string): PackHeader | undefined {
    let parsed: object;
    try {
      parsed = JSON.parse(body) as object;
    } catch (_err) {
      return undefined;
    }
    if (parsed === null || parsed === undefined) {
      return undefined;
    }
    const obj: Record<string, Object> = parsed as Record<string, Object>;
    const ver: Object | undefined = obj.version;
    const schemaVer: Object | undefined = obj.schema_version;
    const words: Object | undefined = obj.words;
    if (!Array.isArray(words)) {
      return undefined;
    }
    const header: PackHeader = new PackHeader();
    header.version = typeof ver === 'number' ? (ver as number) : 0;
    header.schemaVersion = typeof schemaVer === 'number' ? (schemaVer as number) : 1;
    header.wordsLength = (words as object[]).length;
    return header;
  }
}

class PackHeader {
  version: number = 0;
  schemaVersion: number = 1;
  wordsLength: number = 0;
}
```

### Cache topology

- Two LRU buckets (illustrations max 50 entries, audio max 100). Each entry maps `url -> { localPath, lastAccessedMs }`.
- Disk layout: `${context.filesDir}/asset-cache/illustrations/<hash>.<ext>` and `${context.filesDir}/asset-cache/audio/<hash>.<ext>`.
- LRU eviction runs after a successful new write — never mid-render.
- Tests mock the filesystem via an injected `AssetFsAdapter` interface (read / write / delete / list per directory).
- `resolve(url)` returns `Promise<string | undefined>`. When the URL is empty / undefined / fails to download, returns undefined and the consumer falls back to the bundled asset.

```ts
  readonly storage: Map<string, ArrayBuffer> = new Map<string, ArrayBuffer>();
  readonly listings: Map<string, string[]> = new Map<string, string[]>();

  async ensureDir(dir: string): Promise<void> {
    if (!this.listings.has(dir)) {
      this.listings.set(dir, []);
    }
  }
  async exists(path: string): Promise<boolean> {
    return this.storage.has(path);
  }
  async writeBytes(path: string, bytes: ArrayBuffer): Promise<void> {
    if (!this.storage.has(path)) {
      // Track filename in the parent directory listing.
      const parent: string = path.substring(0, path.lastIndexOf('/'));
      const existing: string[] = this.listings.get(parent) ?? [];
      const fileName: string = path.substring(path.lastIndexOf('/') + 1);
      if (existing.indexOf(fileName) < 0) {
        existing.push(fileName);
      }
      this.listings.set(parent, existing);
    }

  }

    const got: ArrayBuffer | undefined = this.storage.get(path);
    if (got === undefined) {
      throw new Error(`FakeFs: no such file ${path}`);
```

```ts
import { http } from '@kit.NetworkKit';
import { fileIo as fs } from '@kit.CoreFileKit';
import { BusinessError } from '@ohos.base';
import { buffer } from '@kit.ArkTS';

export enum AssetKind {
  Illustration = 'illustration',
  Audio = 'audio',
}

export const ASSET_DIR_ILLUSTRATIONS: string = 'illustrations';
export const ASSET_DIR_AUDIO: string = 'audio';
export const ILLUSTRATION_LRU_CAP: number = 50;
export const AUDIO_LRU_CAP: number = 100;
const FETCH_TIMEOUT_MS: number = 6000;

export interface AssetFsAdapter {
  ensureDir(dir: string): Promise<void>;
  exists(path: string): Promise<boolean>;
  writeBytes(path: string, bytes: ArrayBuffer): Promise<void>;
  readBytes(path: string): Promise<ArrayBuffer>;
  deletePath(path: string): Promise<void>;
  listDir(dir: string): Promise<string[]>;

export interface AssetHttpFetcher {
  fetchBinary(url: string): Promise<ArrayBuffer | undefined>;

class IndexEntry {
  url: string = '';
  fileName: string = '';
  lastAccessedMs: number = 0;
}

/**
 * Lazy on-disk LRU cache for word illustration / audio URLs.
 *
 * The cache is keyed by URL; each kind has its own bucket and its
 * own LRU cap. Misses trigger a single fetch via `AssetHttpFetcher`;
 * failures resolve to undefined so the rendering layer can fall
 * back to bundled SVG / TTS without blocking on the network.
 *
 * Tests inject the filesystem adapter and HTTP fetcher; production
 * code uses the `forContext` factory which wires real ohos.file.fs +
 * ohos.net.http implementations.
 */
export class RemoteAssetCache {
  private readonly fsAdapter: AssetFsAdapter;
  private readonly fetcher: AssetHttpFetcher;
  private readonly root: string;
  private readonly indexes: Map<AssetKind, Map<string, IndexEntry>> =
    new Map<AssetKind, Map<string, IndexEntry>>();
  private bootstrapped: boolean = false;

  constructor(fsAdapter: AssetFsAdapter, fetcher: AssetHttpFetcher, root: string) {
    this.fsAdapter = fsAdapter;
    this.fetcher = fetcher;
    this.root = root.replace(/\/+$/g, '');
    this.indexes.set(AssetKind.Illustration, new Map<string, IndexEntry>());
    this.indexes.set(AssetKind.Audio, new Map<string, IndexEntry>());
  }

  static async forContext(ctx: common.UIAbilityContext): Promise<RemoteAssetCache> {
    const fsAdapter: AssetFsAdapter = new RealAssetFs();
    const fetcher: AssetHttpFetcher = new RealAssetFetcher();
    const root: string = `${ctx.filesDir}/asset-cache`;
    return new RemoteAssetCache(fsAdapter, fetcher, root);
  }

  async resolve(url: string, kind: AssetKind): Promise<string | undefined> {
    if (url.length === 0) {
      return undefined;
    }
    await this.ensureBootstrapped();

    const index: Map<string, IndexEntry> = this.indexes.get(kind)!;
    const cached: IndexEntry | undefined = index.get(url);
    if (cached !== undefined) {
      const path: string = this.pathFor(kind, cached.fileName);
      if (await this.fsAdapter.exists(path)) {
        cached.lastAccessedMs = Date.now();
        return path;
      }
      // Disk file vanished; drop the index entry and fall through.
      index.delete(url);
    }

    const bytes: ArrayBuffer | undefined = await this.fetcher.fetchBinary(url);
    if (bytes === undefined) {
      return undefined;
    }
    const fileName: string = RemoteAssetCache.fileNameFor(url);
    const fullPath: string = this.pathFor(kind, fileName);
    await this.fsAdapter.writeBytes(fullPath, bytes);
    const entry: IndexEntry = new IndexEntry();
    entry.url = url;
    entry.fileName = fileName;
    entry.lastAccessedMs = Date.now();
    index.set(url, entry);
    await this.evictIfNeeded(kind);
    return fullPath;
  }

  private async ensureBootstrapped(): Promise<void> {
    if (this.bootstrapped) {
      return;
    }
    this.bootstrapped = true;
    await this.fsAdapter.ensureDir(this.root);
    await this.fsAdapter.ensureDir(`${this.root}/${ASSET_DIR_ILLUSTRATIONS}`);
    await this.fsAdapter.ensureDir(`${this.root}/${ASSET_DIR_AUDIO}`);
    // We deliberately do NOT rebuild url -> fileName mappings from the
    // listing because we don't store URLs on disk. A cold-start
    // process simply re-fetches assets the first time they're
    // requested again — fine for V0.5.D where ≤ 150 small files
    // total, no need to persist the URL index.
  }

  private pathFor(kind: AssetKind, fileName: string): string {
    const bucket: string = kind === AssetKind.Illustration
      ? ASSET_DIR_ILLUSTRATIONS
      : ASSET_DIR_AUDIO;
    return `${this.root}/${bucket}/${fileName}`;
  }

  private async evictIfNeeded(kind: AssetKind): Promise<void> {
    const cap: number = kind === AssetKind.Illustration ? ILLUSTRATION_LRU_CAP : AUDIO_LRU_CAP;
    const index: Map<string, IndexEntry> = this.indexes.get(kind)!;
    if (index.size <= cap) {
      return;
    }
    const ordered: IndexEntry[] = [];
    index.forEach((e: IndexEntry): void => { ordered.push(e); });
    ordered.sort((a: IndexEntry, b: IndexEntry): number => a.lastAccessedMs - b.lastAccessedMs);
    const overflow: number = index.size - cap;
    for (let i = 0; i < overflow; i++) {
      const victim: IndexEntry = ordered[i];
      const path: string = this.pathFor(kind, victim.fileName);
      try {
        await this.fsAdapter.deletePath(path);
      } catch (err) {
        const be: BusinessError = err as BusinessError;
        console.error(`RemoteAssetCache.evict failed for ${path}: ${JSON.stringify(be)}`);
      }
      index.delete(victim.url);
    }
  }

  private static fileNameFor(url: string): string {
    const dotIdx: number = url.lastIndexOf('.');
    const ext: string = dotIdx > 0 ? url.substring(dotIdx + 1).toLowerCase() : 'bin';
    const safeExt: string = ext.length > 5 ? 'bin' : ext;
    const hash: string = RemoteAssetCache.hashUrl(url);
    return `${hash}.${safeExt}`;
  }

  private static hashUrl(url: string): string {
    // FNV-1a 32-bit hash (no dependency on @ohos.security.cryptoFramework
    // — overkill for a non-security cache key). Hex-encoded, 8 chars.
    let h: number = 0x811c9dc5;
    for (let i = 0; i < url.length; i++) {
      h ^= url.charCodeAt(i);
      h = (h * 0x01000193) >>> 0;
    }
    return h.toString(16).padStart(8, '0');
  }
}

class RealAssetFs implements AssetFsAdapter {
  async ensureDir(dir: string): Promise<void> {
    try {
      await fs.mkdir(dir, true);
    } catch (err) {
      const be: BusinessError = err as BusinessError;
      // 13900015 == file exists; safe to ignore. Other codes bubble.
      if (be.code !== 13900015) {
        throw err;
      }
    }
  }
  async exists(path: string): Promise<boolean> {
    try {
      return await fs.access(path);
    } catch (_err) {
      return false;
    }
  }
  async writeBytes(path: string, bytes: ArrayBuffer): Promise<void> {
    const file: fs.File = await fs.open(path, fs.OpenMode.READ_WRITE | fs.OpenMode.CREATE | fs.OpenMode.TRUNC);
    try {
      await fs.write(file.fd, bytes);
    } finally {
      await fs.close(file);
    }
  }
  async readBytes(path: string): Promise<ArrayBuffer> {
    const file: fs.File = await fs.open(path, fs.OpenMode.READ_ONLY);
    try {
      const stat: fs.Stat = await fs.stat(path);
      const bufRef: ArrayBuffer = new ArrayBuffer(stat.size);
      await fs.read(file.fd, bufRef);
      return bufRef;
    } finally {
      await fs.close(file);
    }
  }
  async deletePath(path: string): Promise<void> {
    try {
      await fs.unlink(path);
    } catch (err) {
      const be: BusinessError = err as BusinessError;
      if (be.code !== 13900002) { // ENOENT — already gone, fine.
        throw err;
      }
    }
  }
  async listDir(dir: string): Promise<string[]> {
    try {
      return await fs.listFile(dir);
    } catch (_err) {
      return [];
    }
  }

class RealAssetFetcher implements AssetHttpFetcher {
  async fetchBinary(url: string): Promise<ArrayBuffer | undefined> {
    const req: http.HttpRequest = http.createHttp();
    try {
      const resp: http.HttpResponse = await req.request(url, {
        method: http.RequestMethod.GET,
        connectTimeout: FETCH_TIMEOUT_MS,
        readTimeout: FETCH_TIMEOUT_MS,
        expectDataType: http.HttpDataType.ARRAY_BUFFER,
      });
      if (resp.responseCode !== 200) {
        return undefined;
      }
      if (resp.result instanceof ArrayBuffer) {
        return resp.result;
      }
      // Some emulator builds normalise binary into Uint8Array. Convert.
      if (resp.result instanceof Uint8Array) {
        return (resp.result as Uint8Array).buffer.slice(0);
      }
      // String fallback (shouldn't happen with expectDataType=ARRAY_BUFFER
      // but cheap belt-and-braces): UTF-8 encode.
      if (typeof resp.result === 'string') {
        return buffer.from(resp.result as string, 'utf-8').buffer;
      }
      return undefined;
    } catch (err) {
      const be: BusinessError = err as BusinessError;
      console.error(`RealAssetFetcher.fetchBinary failed for ${url}: ${JSON.stringify(be)}`);
      return undefined;
    } finally {
      req.destroy();
    }
  }
}
```

```ts
export class CategoryRow {
  id: string = '';
  labelEn: string = '';
  labelZh: string = '';
  storyZh?: string;
}

interface RawCategory {
  id?: string;
  labelEn?: string;
  labelZh?: string;
  storyZh?: string;
}

interface RawPack {
  categories?: RawCategory[];
}

/**
 * V0.5 schema_v4: keeps the pack.categories[] block in memory so
 * HomePage can overlay server storyZh / labelZh on top of the
 * bundled AdventureCatalog regions whenever the operator has
 * synced a v4+ pack. Pure static parser plus a tiny lookup map —
 * no IO, no AppStorage, owned by HomePage's lifecycle.
 */
export class CategoryCatalog {
  private rows: CategoryRow[] = [];
  private byIdMap: Map<string, CategoryRow> = new Map<string, CategoryRow>();

  setRows(rows: CategoryRow[]): void {
    this.rows = rows.slice();
    this.byIdMap = new Map<string, CategoryRow>();
    for (let i = 0; i < this.rows.length; i++) {
      this.byIdMap.set(this.rows[i].id, this.rows[i]);
    }
  }

  getById(id: string): CategoryRow | undefined {
    return this.byIdMap.get(id);
  }

  size(): number {
    return this.rows.length;
  }

  static parsePackCategories(packJson: string): CategoryRow[] {
    if (packJson.length === 0) {
      return [];
    }
    let parsed: RawPack;
    try {
      parsed = JSON.parse(packJson) as RawPack;
    } catch (_err) {
      return [];
    }
    if (parsed === null || parsed === undefined || parsed.categories === undefined) {
      return [];
    }
    const out: CategoryRow[] = [];
    const arr: RawCategory[] = parsed.categories;
    for (let i = 0; i < arr.length; i++) {
      const r: RawCategory = arr[i];
      if (typeof r.id !== 'string' || r.id.length === 0) {
        continue;
      }
      const row: CategoryRow = new CategoryRow();
      row.id = r.id;
      row.labelEn = typeof r.labelEn === 'string' ? r.labelEn : '';
      row.labelZh = typeof r.labelZh === 'string' ? r.labelZh : '';
      if (typeof r.storyZh === 'string' && r.storyZh.length > 0) {
        row.storyZh = r.storyZh;
      }
      out.push(row);
    }
    return out;
  }
}
```

```ts
import { router } from '@kit.ArkUI';
import { BusinessError } from '@ohos.base';
import { AdminApiClient, AdminAuthError } from '../../services/AdminApiClient';
import { SERVER_BASE_URL } from '../../services/RemoteWordPackConfig';

const ADMIN_JWT_KEY: string = 'admin_jwt';

@Entry
@Component
struct AdminLoginPage {
  @State private username: string = '';
  @State private password: string = '';
  @State private busy: boolean = false;
  @State private toast: string = '';
  private toastTimer: number = -1;

  private async submit(): Promise<void> {
    this.busy = true;
    try {
      const client: AdminApiClient = new AdminApiClient(/* real adapter */, SERVER_BASE_URL);
      const token: string = await client.login(this.username, this.password);
      AppStorage.setOrCreate<string>(ADMIN_JWT_KEY, token);
      await router.replaceUrl({ url: 'pages/admin/AdminWordsPage' });
    } catch (e) {
      if (e instanceof AdminAuthError) {
        this.flash('登录失败:用户名或密码错误');
      } else {
        this.flash(`登录失败:${(e as BusinessError).message}`);
      }
    } finally {
      this.busy = false;
    }
  }

  private flash(text: string): void {
    this.toast = text;
    if (this.toastTimer >= 0) {
      clearTimeout(this.toastTimer);
    }
    this.toastTimer = setTimeout((): void => {
      this.toast = '';
      this.toastTimer = -1;
    }, 2400);
  }

  build() {
    Column({ space: 12 }) {
      Text('管理员登录').fontSize(22).fontWeight(FontWeight.Bold);
      TextInput({ placeholder: 'username' })
        .id('AdminUsernameInput')
        .onChange((v: string): void => { this.username = v; });
      TextInput({ placeholder: 'password' })
        .id('AdminPasswordInput')
        .type(InputType.Password)
        .onChange((v: string): void => { this.password = v; });
      Button(this.busy ? '登录中…' : '登录')
        .id('AdminLoginSubmit')
        .enabled(!this.busy)
        .onClick((): void => { this.submit(); });
      if (this.toast.length > 0) {
        Text(this.toast).id('AdminLoginToast').fontColor('#E63946');
      }
    }
    .padding(20)
    .width('100%')
    .height('100%');
  }
}
```
