# V0.4.1 Spelling Question Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans for inline execution.

**Spec:** [`docs/superpowers/specs/2026-04-29-v0.4.1-spelling-question-design.md`](../specs/2026-04-29-v0.4.1-spelling-question-design.md)

**Goal:** Boss-only completed spelling question (scrambled letter pool, first-letter revealed, shake-red on wrong taps, no HP cost, no consume).

**Architecture:** New `QuestionKind.Spell`, new `SpellGenerator` service, new `SpellingArea` component owning per-tap state locally. PlanQuestionSource splits Boss from Elite (Boss inserts SpellGenerator before existing FillLetterMedium fallback). BattleEngine.submitAnswer contract unchanged — SpellingArea only fires it once when all blanks are filled.

**Tech Stack:** ArkTS, ArkUI, Hypium tests, HarmonyOS NEXT.
